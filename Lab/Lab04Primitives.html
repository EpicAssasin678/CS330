TRIANGLE_STRIP has issues with the lower-half of this render due to the fact that it is connecting triangles using context for triangle a using two fo the previous triangles points b, creating the oblique incision and other triangles from not really rendering. TRAINGLE_FAN has it's own issues. For example two trangles sharing just one point to make a sort of "jagged-teeth" look would be imposible as the initial point p1 would instead connect allo other points in respect to itself.
